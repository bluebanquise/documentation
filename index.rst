BlueBanquise Documentation
==========================

.. image:: images/BlueBanquise_logo_dark.svg
   :align: center

Welcome.

You should find here all needed 
information on how to deploy and use the BlueBanquise stack.

------------

These pages contain information on:

* How to boostrap BlueBanquise.
* How to deploy a basic cluster.
* How to deploy an advanced cluster.
* How to deploy a very large cluster.
* How to use some community roles to enhance and specialize cluster.
* How to use BlueBanquise as an Ansible collection of roles.

This documentation also contains few tutorials, for generic knowledge:

* How to manually deploy a test HPC cluster (generic knowledge, not related to the stack)
* How to use Ansible (generic knowledge, not related to the stack)

------------

issues in this documentation.
